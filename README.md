# 炫酷鼠标拖尾效果网站

这是一个展示炫酷鼠标拖尾效果的静态网站。使用 HTML5 Canvas 实现流畅的拖尾动画效果。

## 特点

- 流畅的鼠标拖尾效果
- 物理引擎模拟的惯性效果
- 渐变色彩和发光效果
- 响应式设计
- 故障艺术文字效果

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API

## 本地运行

1. 克隆仓库
```bash
git clone [仓库地址]
```

2. 打开 `index.html` 文件即可在浏览器中查看效果
